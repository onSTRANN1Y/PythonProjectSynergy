# Калькулятор факториала

Программа запрашивает у пользователя положительное целое число, проверяет корректность ввода и вычисляет его факториал.

## Функционал

- Запрос положительного целого числа у пользователя.
- Валидация ввода: проверка на целочисленность и положительность.
- Обработка ошибок: понятные сообщения при вводе нечисловых данных, нуля или отрицательных чисел.
- Повторный запрос ввода при ошибке без завершения программы.
- Вычисление факториала с использованием оптимизированной функции `math.factorial`.

## Как запустить

1. Убедитесь, что установлен Python 3.8+.
2. Сохраните файлы `factorial.py`, `.gitignore` и `README.md` в одну папку.
3. Запустите программу:
   ```bash
   python factorial_solver.py
